# Quarentena de limpeza

Esta pasta registra itens classificados como duvidosos na limpeza.

## Itens duvidosos mantidos no projeto

- `rd-debug-dump.html`
- `js/rd-debug-dump.js`

Motivo: ferramenta de diagnostico local que pode ser usada em suporte.

## Validacao executada

- Preservados todos os arquivos de operacao (API, dashboards, visitador, scripts, dados e uploads).
- Removidos apenas itens de baixo risco (logs, cache temporario, cookie local e patch one-shot).
