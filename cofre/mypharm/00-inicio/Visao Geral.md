# Visão Geral — MyPharm

## O que é

Sistema web interno de uma **farmácia de manipulação** para gestão comercial, acompanhamento de visitadores médicos e análise de prescritores.

## Para que serve

| Área | Funcionalidade |
|---|---|
| Gestão | KPIs de vendas, metas, ranking de vendedores |
| Vendedor | Pedidos, comissões, revendas |
| Visitador | Rotas GPS, prescritores, bonificações |
| TV | Painel ao vivo gamificado para a equipe |
| Executivo | Painel de gestão comercial, funil, comparativos |

## Quem usa

- **Admin / Gestor** — acesso total, configurações
- **Vendedor** — vê seus pedidos, comissões, ranking
- **Visitador** — registra visitas, prescritores, rotas
- **Gerente** — painel executivo, aprovação de revendas

## Onde roda

| Ambiente | URL | Banco |
|---|---|---|
| Local (dev) | `http://localhost/mypharm/` | XAMPP MySQL local |
| Produção | Hostinger | `u936212550_my_pharm` |

## Timezone

`America/Porto_Velho` (UTC-4) — usado em todas as queries e datas do sistema.
